# Ecommerce Sales Analysis 📈

## Project Overview
This project analyzes E-commerce sales data to identify key performance indicators (KPIs), customer behavior trends, and revenue growth.

## Key Insights
* **Sales Trends:** Analysis of monthly and quarterly sales growth.
* **Customer Segments:** Identification of high-value customer groups.
* **Product Performance:** Top-selling categories and profit margins.

## Tools Used
* **Excel:** Data Cleaning and Analysis.
* **Power BI:** Data Visualization and Dashboarding.
* **Power Query:** ETL process.
* ## 🛠️ Data Processing (ETL)
* Cleaned raw data using **Power Query** (Removed duplicates, handled null values).
* Created **Date Table** for Time Intelligence analysis.
* Performed Data Modeling with 1-to-many relationships.

## 📊 Business Metrics Analyzed
* **Total Sales & Profit:** Overall financial performance.
* **MoM (Month-over-Month) Growth:** Tracking sales consistency.
* **Top 5 Products:** Best-selling items by revenue and quantity.
* **Regional Performance:** Sales distribution across different cities/states.
